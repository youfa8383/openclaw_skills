# 更新日志

所有显著更改都将记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
并且本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.2.0] - 2026-04-01

### 修复
- **重大修复**: 移除有问题的 `System.Windows.Forms.Screen` 依赖
- **分辨率问题**: 使用固定的1920×1080分辨率，避免截图不完整
- **Bitmap构造函数**: 修复 `New-Object System.Drawing.Bitmap` 问题
- **错误处理**: 添加完整的异常捕获和JSON输出

### 新增
- **新脚本**: `screenshot_simple.ps1` - 简单可靠的修复版本
- **安装脚本**: `install.sh` - 一键安装
- **GitHub支持**: 完整的仓库结构和文档
- **测试验证**: 实际测试通过，用户确认正确

### 变更
- **版本升级**: v1.0.0 → v1.2.0
- **文档更新**: 完整的README和技能文档
- **文件清理**: 移除测试截图和临时文件

## [1.0.0] - 2026-04-01

### 新增
- **初始版本**: 基于PowerShell的系统截图工具
- **核心功能**: 截图捕获和消息平台集成
- **技能结构**: 完整的OpenClaw技能文件
- **基础文档**: SKILL.md和使用示例

### 已知问题
- 截图分辨率不正确（只捕获1280×720区域）
- `System.Windows.Forms.Screen` 无法正确工作
- 缺少完整的错误处理

- **次版本号 (x.2.x)**: 向下兼容的功能性新增
- **修订号 (x.x.0)**: 向下兼容的问题修正
